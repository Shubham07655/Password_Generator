# cipher - Password Generator

A sleek and modern password generator with a beautiful glass-morphism UI. Created by Shubham Yadav.

![password generator preview]

## Features

- 🎨 **Beautiful UI** - Glass morphism design with smooth animations
- 🔐 **Secure Passwords** - Generates strong, random passwords
- 📊 **Strength Meter** - Visual indicator of password strength
- ⚙️ **Customizable** - Choose length and character types
- 📋 **Copy to Clipboard** - One-click copy with toast notification
- 📱 **Responsive** - Works on all devices

## Character Options

- Uppercase letters (A-Z)
- Lowercase letters (a-z)
- Numbers (0-9)
- Symbols (!@#$%^&*()_+[]{}|;:,.<>?)

## How to Use

1. Adjust the slider to set password length (4-30 characters)
2. Check/uncheck character type options
3. Click "generate" or "shuffle" to create a new password
4. Click the copy button to copy to clipboard

## Technologies Used

- HTML5
- CSS3 (with modern features like backdrop-filter, gradients)
- JavaScript (ES6+)
